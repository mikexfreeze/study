“光影·沉浸 体验馆(Hack the soul)”为2016缤智中国先锋艺术项目的主体，体验馆以 先锋艺术光影数字化装置互动体验为核心，延展为线上线下两种展览形式，以数字艺术 中最为常见的“光影”，多角度来陈述与探索缤智的设计和技术关键。 今年的“光影·沉浸 体验馆”以缤智多变外观和钻石切面为设计概念，营造先锋的艺术空间， 以钻石切面的几何视觉切入，以具有科技、先锋感的策展角度，突显缤智视觉元素和展区的 视觉效果，将各个展区以“情节化”有机串联，就恰如缤智英文名“VEZEL”所寄寓的， 以锻造BEZEL（宝石切割面）的手法恰如其分地打造一款在不同角度都闪耀超凡魅力 的Vehicle（车辆）。

时间：

2015年7月-10月

主题形式：

以先锋艺术之名，联动艺术家，结合车型卖点打造的先锋艺术装置互动体验

内容：

2015年缤智中国先锋艺术项目第一届以 “即行变奏”（Hack the city）为主题强力开启，展期自7月 开始巡展北京、成都、广州三地，直至10月份结束，邀请了全国著名从事表演、影像、声音、数字、互 动等跨媒体艺术的艺术家8GG、徐文恺和吴超的参与，共同探讨当代都市的生活，包括城市文明、移动 生活等话题。 展览跳出过往美术馆的展示方式，把艺术展搬到公共空间，公众通过与艺术互动获得全新体验，建立对 当代艺术的认知，同时也支持鼓励未来中国年轻艺术家的创作。

尚雯婕

2016缤智中国先锋艺术项目特邀电子唱作人尚雯婕担任本次项目的音乐总监。
这次跨界合作是尚雯婕首次以音乐总监的身份参与到汽车艺术项目， 也让2016缤智中国先锋艺术项目成为国内第一个拥有音乐总监的汽车艺术巡展。

速动流淌于时光激流隧道

以缤智（VEZEL）的LED投影式前大灯和钻石般多变的外观为创作灵感，打造钻石切面体感的光影互动体感隧道，令观众入场即有奇炫无比的视觉体验。

沸腾蔓延于未来光感动力

以城市人文情怀来装点迷宫般的幕布装置，结合光电的效果，让观众犹如驾驶缤智（VEZEL）穿梭于城市光影中，如梦似幻。

颤动置身于宇宙方程赛道

透过缤智的梦幻科技感航空式座舱，用沉浸式视听感官将缤智车变为光影体验舱。透过缤智全景车窗看到未来时光隧道中的钻石般几何光影，声光影联动呼应将感官体验升级，与缤智穿越时空，感受缤智的先锋魅力。

图腾映照于超越时空幻想

以数字AR技术为主，通过逆时空遁入的形式展现缤智（VEZEL）钻石般时尚外观和超动感核心动力，让观众获得前所未有的新奇和梦幻体验。
